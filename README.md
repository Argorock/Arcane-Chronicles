## Instructions for Build and Use

Steps to build and/or run the software:

1. downlaod Godot 4.x from the godot website
2. clone this repository to your computer
3. open godot, select import, and choose the folder containing the project.godot folder
4. once the project loads, click on the run project button in the top right corner
5. if godot ask to set a main scene, choos the scenes/MainMenu.tscn

Instructions for using the software:

1. start the game by clicking on the run project button
2. use the in game ui to change scenes, click on load game for now
3. use the WASD keys to move, and E to interact with things, SPACEBAR also works to jump if you prefer that
